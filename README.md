- Mini compilador em Python

O projeto é um simples compilador em Python.
Ele ler tipo uma linguagem personalizado e traduz código para Python.

Exemplo de entrada:
let x = 1972
let y = 5
print x + y

Código Python apresentado:
x = 1972
y = 5
print(x + y)

Recursos que foram colocados:
- Variáveis
- Operações matemáticas (+ - * /)
- Impressão de expressões
- Estrutura básica de um compilador (analisador léxico, analisador sintático, compilador)
